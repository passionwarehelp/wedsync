# WedSync Complete File Structure Guide

This document shows EXACTLY where every file should be placed in your GitHub repository.

## Root Level Files (/)

These files go in the **root** of your repository:

```
/
├── .env                          # Environment variables (DO NOT commit to GitHub - add to .gitignore)
├── .gitignore                    # Git ignore file
├── .prettierrc                   # Prettier config
├── App.tsx                       # Main app entry point
├── AUTHENTICATION_SETUP_GUIDE.md # Auth setup documentation
├── CLAUDE.md                     # Claude instructions
├── QUICK_START.md                # Quick start guide
├── README.md                     # Project documentation
├── app.json                      # Expo configuration
├── babel.config.js               # Babel configuration
├── bun.lock                      # Bun lockfile
├── eslint.config.js              # ESLint configuration
├── expo.log                      # Expo logs (add to .gitignore)
├── generate-asset-script.ts      # Asset generation script
├── global.css                    # Global styles
├── index.ts                      # App entry point
├── metro.config.js               # Metro bundler config
├── nativewind-env.d.ts           # NativeWind TypeScript definitions
├── package.json                  # Dependencies
├── tailwind.config.js            # Tailwind configuration
└── tsconfig.json                 # TypeScript configuration
```

## Summary

**Total Structure:**
- **Root**: 19 configuration and entry files
- **/assets**: Images and media
- **/backend**: Separate backend server (Prisma + Hono)
- **/cloudflare-workers**: Serverless functions
- **/patches**: Package patches
- **/src**: Main mobile app code
  - **/api**: 7 API integration files
  - **/components**: Reusable components
  - **/lib**: Utilities and hooks
  - **/navigation**: Navigation setup
  - **/screens**: 30+ screen components
  - **/state**: 4 Zustand stores
  - **/types**: TypeScript definitions
  - **/utils**: Helper functions

**Everything in this workspace is production-ready React Native code for the WedSync mobile app.**
